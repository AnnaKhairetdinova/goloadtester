# goloadtester

GoLoadTester - это CLI-утилита для нагрузочного тестирования HTTP-эндпоинтов.
Отправляет N запросов на указанный URL с заданной конкурентностью и показывает статистику: RPS, латентность (включая P50/P90/P99), распределение статус-кодов. 

Что сдлеано:
- конкурентность через горутины и каналы
- синхронизация через WaitGroup
- корректный сбор статистики из нескольких горутин
- читаемый CLI через флаги
- покрытие тестами

Пример запуска: go run main.go -url https://ya.ru/ -n 100 -c 50 -timeout 5s
Флаги:
- -url URL для запроса 
- -n Количество запросов 
- -c Количество параллельных запросов 
- -timeout Время запроса

Остановка: нажми Ctrl+C для получения частичного отчета
